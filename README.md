#### Kazam as a Screen Recorder
    Super+Ctrl+R: Start recording  
    Super+Ctrl+P: Pause recording, press again for resuming the recording  
    Super+Ctrl+F: Finish recording  
    Super+Ctrl+Q: Quit recording
